## 5. `05-plugin.md`

放 uts 插件和原生能力规则。

```md
# UTS Plugin Rules

## 涉及以下能力时必须先做兼容性评估

- 扫码
- 定位
- 蓝牙
- 推送
- 支付
- 地图
- 相机
- 相册
- 文件选择
- 第三方 SDK
- 系统权限
- 设备信息

## 强制规则

- 优先使用 uts 插件。
- 禁止未经验证直接引入传统 uni-app native plugin。
- 插件必须验证 Android、iOS、H5/Web、小程序目标端。
- 插件 API 必须封装，不允许页面直接调用底层插件复杂逻辑。

## 验证清单

```text
[ ] 是否支持 uni-app x
[ ] 是否支持 Android
[ ] 是否支持 iOS
[ ] 是否支持 Web/H5
[ ] 是否支持目标小程序
[ ] 是否影响云打包
[ ] 是否需要额外权限
[ ] 是否有失败回调
[ ] 是否有维护风险
```
