# UI 占位实现说明

此目录用于存放当前阶段的桥接实现说明与内部约定。

当前仓库已真实导入官方 `uni-nav-bar-x`。后续若继续通过 HBuilderX 插件市场或官方开源仓库导入真实组件，请将真实组件保留在项目根目录的 `uni_modules/` 中，再由 `components/ui/*` 桥接组件按需切换内部实现。

## 当前桥接关系

- `UiNavBar`
  - 当前实现：桥接到真实 `uni-nav-bar`
  - 后续策略：继续保持桥接收口
- `UiSectionCard`
  - 当前实现：仓库内自研通用卡片
  - 未来策略：默认保持自研，按需组合真实组件能力
- `UiEmptyState`
  - 当前实现：仓库内自研空态组件
  - 未来策略：默认保持自研，必要时与真实组件组合

## 约束边界

- 仅引入可追溯到官方来源的真实 `uni-ui x` 源码
- 不在业务页面中直接写死未来插件路径
- 演示页与文档状态需与仓库真实接入状态保持一致
